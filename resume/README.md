# Resume Workspace

## Folder Structure

- `drafts/`: working drafts that are still being refined
- `notes/`: interview logs, project context, and source notes
- `outputs/`: ready-to-use deliverables for LinkedIn or portfolio
- `references/`: external reference materials such as sample PDFs

## Recommended Usage

- Start from `outputs/` when you need copy-ready English source text.
- Check `drafts/` when you want to revise wording, layout, or emphasis.
- Use `notes/` to trace the original rationale, project context, or interview answers.
- Use `references/` only as structural inspiration, not as content to copy.
