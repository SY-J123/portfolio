가설 수립

산출물 과다와 내용 부정확이라는 문제에 대해, 산출물을 '적게·정확히·빠르게' 작성할 수 있도록 두 가지 방향의 가설을 세웠다.

가설 1

산출물을 감소·효율화하고 시각 자료로 기준을 명확히 하면 각 산출물에 집중할 수 있어, 더 정확한 내용을 더 빠르게 작성할 수 있을 것이다.

연관 이슈
  • 산출물 과다
  • 작업 기준 불명확
목표
  • 기획 내용 정확도 향상
  • 기획 작성 시간 단축
평가방안
  • 와이어프레임 제거 및 바이브코딩 프로토타입 전환
  • IA와 서비스 흐름도 병합
  • UI 기능 정의서 분할 작성

가설 2

산출물을 하나의 장소에 저장하고 AI로 유지보수를 자동화하면 유지보수 부담이 줄어들 것이다.

연관 이슈
  • 문서 정합성 저하
  • 정합성 저하
목표
  • 모든 문서를 하나의 플랫폼에서 관리
