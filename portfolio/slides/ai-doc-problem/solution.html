1

와이어프레임 제거 + 프로토타입 전환

바이브코딩으로 프로토타입을 직접 제작해 와이어프레임보다 시각적으로 더 풍부한 화면과 간단한 인터랙션을 함께 전달하면서, 작업 기준이 불명확했던 부분을 보완했다. 동시에 와이어프레임 작성을 제거해 작성 시간을 단축했다.

2

IA를 서비스 흐름도로 병합

IA와 서비스 흐름도는 유사한 내용이 많아 하나의 구조 문서로 병합해 관리했다.

3

UI 기능 정의서 분할 작성

기능 위주로 작성하고, 데이터 등 구체적인 내용은 제작 완료 후 AI로 자동 정리해 작성 시간을 줄였다.
