부록 — 대기업 vs LULU.AI 프로세스

대기업 재직자 인터뷰로 담당자별 프로세스를 파악한 뒤 우리 조직과 비교한 내용. 산출물 종류는 비슷하지만 담당자 수가 적어 한 사람당 담당 범위가 넓다는 구조적 차이를 확인했다.

대기업 프로세스

사업 담당자

주요 역할
사업 목표와 요구사항을 정의한다.
주요 산출물
사업 요구사항, 목표, 우선순위

서비스 기획자

주요 역할
서비스 정책과 기능 구조를 설계한다.
주요 산출물
서비스 정책, 서비스 기능 정의서, IA, 서비스 흐름도

UI/UX 기획자

주요 역할
사용자 흐름과 화면 구조를 설계한다.
주요 산출물
UI 규격서, UI 기능 정의서, 와이어프레임

디자이너

주요 역할
화면 시각화와 인터랙션을 설계한다.
주요 산출물
디자인 시안, 디자인 화면 가이드, 프로토타입
